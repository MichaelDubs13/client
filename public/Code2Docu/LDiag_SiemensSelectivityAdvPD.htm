Functional description

DOCU

Jira: ACE-28117: LDiag_SiemensSelectivityAdvPD

This FB and associated HMI faceplate supports reading diagnostic information for the SEL1200 and SEL1400 series of Siemens Selectivity Modules (24V electronic circuit breakers) as well as remotely resetting an output channel electrical fault without needing to access the physical hardware. The FB can be configured to raise ProDiag alarms if any of the 24V output channels of the selectivity module are not ON or alarm if there is an issue with the module itself.

Data read from the module includes:

  • State of the DIP switches (used to enable IO diagnostics and change the startup behavior).
  • Type (SEL1200 v SEL1400)
  • Article number (6EP4438-7FB00-3DX0)
  • Serial number (O6/R8)
  • Module status (OK, overvolt/undervolt, overcurrent, etc)
  • For each output channel: status, actual current, adjustable current limit setpoint, state, and reason an output channel is switched off.

Data written to the module:

  • RESET command for restoring 24V current for all channels in the 'automatically switched off, reset possible state'.
    • Note: RESET does not energize outputs that are manually switched off.

Hardware Setup

Visit the SIEMENS SITOP SEL1200-SEL1400 Manual for detailed installation and use information.

  1. Connect the PLC DI to the module signal contact (pins 13/14).
  2. Connect the PLC DQ to the module reset input.
  3. Set the COM DIP switch to '1' to enable diagnostics.
  4. Set the TD1/TD2 switches as required to '1' to enable a time delay between outputs being switched on.

PLC Program Integration

  1. Create cyclic interrupt OB with a 10ms cyclic time (10000us)
  2. Assign the input module associated with the selectivity module diagnostic input tag to the Cyclic Interrupt OB and process image partition (PIP).
  3. Instantiate the FB (LDiag_SiemensSelectivityAdvPD) inside the cyclic interrupt OB and configure the block inputs/outputs.

HMI Components

HMI Tags

Wincc Faceplate The faceplate dynamically resizes to show either 4 or 8 outputs depending on which model of selectivity module has been connected, and displays the configured module name (plant identifier), and the name and status of each output. The status indicator matches the LED colors shown on the selectivity module.

HMI Faceplate

Selectivity Module Color Key

HMI Faceplate Interface Tag Setup

Wincc Popup The pop-up screen provides more diagnostic information about the module and offers the ability to reset a channel fault from the HMI. Note that the RESET button only resets output channels that have been automatically switched off AND reset is possible. The RESET button will not enable output channels that have been manually switched off - those must be manually switched back on at the module.
